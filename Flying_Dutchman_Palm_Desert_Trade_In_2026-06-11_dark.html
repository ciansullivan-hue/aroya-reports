v2 correction (2026-06-11)
AROYA's trade-in credit is tied to the SKU purchased, not the device traded in. Each TEROS One Single bought = $100 off ($419 → $319). Each TEROS One Dual bought = $150 off ($608 → $458). Each Climate One bought = $150 off ($569 → $419). So the customer's total credit depends on how many of each new SKU they buy — and the Dual SKU lets them retire 2 legacy pucks for the price of less than 1.5 Singles with the same per-unit discount as a Climate One. That trade-off is the core decision in this report.

Estimated values (flagged E in tables — NOT AROYA-published)

The "Estimated residual value" column reflects defensible book value of installed hardware based on industry-standard 5-year straight-line depreciation, with a health haircut for any device in failure state. Legacy node radio = $280, repeater = $320, TEROS-12 sensor = $140, ATM14 = $180. None of these are AROYA-published — they're informational only and don't feed the trade-in credit math.

Warranty rule

Climate One / H421: 1y standard + 2y CO2. H311: 3y. Everything else: 1y. Warranty start = provisionedOn if present, else createdOn.
